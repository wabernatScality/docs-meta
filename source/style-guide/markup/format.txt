===========
Text Markup
===========

.. include:: /includes/styles/corrections.rst

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

{+docsteam+} follows these conventions in writing {+rst+} and |yaml|:

.. _PEP-008: https://www.python.org/dev/peps/pep-0008/#id19

.. list-table::
   :widths: 20 40 40
   :stub-columns: 1
   :header-rows: 1

   * - Setting
     - Standard
     - Reason
   * - Character Set
     - ASCII
     - Giza and Sphinx can return errors from text that uses higher
       level Unicode characters, including non-printing ones.
   * - Line length
     - Hard breaks at 72 characters

       Exceptions include:

       - :doc:`code examples </style-guide/markup/directives/code-blocks>`
       - :doc:`cross references </style-guide/markup/roles/cross-ref>`

     - Per PEP-008_ and Code Review, tool does not text-wrap.
   * - Line endings
     - UNIX (LF)
     - macOS and Unix (Linux) end lines with line feeds. Using this
       form eliminates rendering errors in your text editor.
   * - Tabs
     - Use 2 spaces per tab.
     -
       - YAML cannot use tabs.
       - PEP-008_ specifies spaces.
   * - End of File
     - Include a blank line.
     - All cross-references are added to the bottom of each assembled {rst+}
       page. Omitting that extra space results in errors.
   * - Variable and File Naming Convention
     - kebab-case
     - Existing MongoDB convention.
   * - Indents
     - As tabs, except when after a directive, then three spaces.
     - {rst+} follows PEP-008 conventions: text blocks must align to
       the same indent level.

.. sidebar:: Available plugins that support hard wraps

   .. list-table::
      :widths: 20 80
      :stub-columns: 1

      * - VSCode
        - `Rewrap <https://marketplace.visualstudio.com/items?itemName=stkb.rewrap>`__
      * - Sublime Text
        - `Wrap Plus <https://packagecontrol.io/packages/Wrap%20Plus>`__

Spaces
------

- Use one space between sentences.

- Place spaces between nested parentheticals and elements in
  JavaScript examples.

  .. list-table::
     :header-rows: 1
     :widths: 50 50

     * - Use
       - Avoid

     * -

         .. code-block:: JavaScript

            { [ a, a, a ] }

       -

         .. code-block:: JavaScript
            :copyable: false

            {[a,a,a]}
